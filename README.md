# kouluta-oma-neuroverkko
Yksinkertainen Google Colab notebook BERT-pohjaisen luokittelijan kouluttamiseen.

Ohjeet:
1. Klikkaa auki `kouluta_oma-neuroverkko_ipynb`.
2. Klikkaa tiedoston vasemmassa yläkulmassa olevaa painiketta `Open in Colab`
3. Tiedosto avautuu Google Colab ympäristössä. Sinun pitää kirjautua Google-tilillesi ennen kuin voit ajaa koodia.
